###  **项目介绍**：  
      一个适用于 Pocketmine-MP 的简单的防止创造利用某些物品卡BUG的一个反作弊插件，我们会持续更新使之成为一款适配MCPE服务器的反作弊插件！
      -可用功能：  
       - [√] 添加配置文件，可选择功能是否开启  
       - [√] 炼药锅创造放入无限药水  
       - [√] 漏斗无限复制物品 
       - [√] 修复创造玩家使用物品展示框卡物品BUG  
       - [√] 通过指令自定义添加/删除管理员，查看列表
       - [√] 修复创造利用工作台合成获得创造不允许使用的物品
      -计划功能：  
       - [x] 更多的自定义  
       - [x] 生存作弊部分拓展
